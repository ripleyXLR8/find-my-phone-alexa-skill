ℹ️ Comment obtenir mon Device ID et mon Secret ?

Pour des raisons de sécurité (Anti-Bot et Double Authentification), Google bloque les connexions automatisées sur les serveurs NAS. Vous devez générer ces codes une seule fois depuis votre ordinateur personnel (Windows/Mac/Linux).

Étape 1 : Préparation
Sur votre PC, installez Python et Git.
Étape 2 : Lancement de l'outil
Ouvrez un terminal (ou Invite de commandes) sur votre PC et tapez une par une : 
git clone https://github.com/leonboe1/GoogleFindMyTools.git
cd GoogleFindMyTools
pip install -r requirements.txt
python main.py
Étape 3 : Connexion
Une fenêtre Google Chrome va s'ouvrir. Connectez-vous avec le compte Google lié au téléphone que vous souhaitez faire sonner.
Étape 4 : Copier les infos
  • Device ID : Regardez dans votre terminal, l'outil affichera la liste de vos appareils avec leurs IDs. Copiez l'ID de votre téléphone.
  • Secret JSON : Allez dans le dossier GoogleFindMyTools/Auth/ sur votre PC. Ouvrez le fichier secrets.json avec le Bloc-notes, et copiez TOUT le texte à l'intérieur.
